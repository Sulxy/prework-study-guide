HTML

  • The head element contains information about the webpage.
  • The body element represents the visible content shown to the user.
  • ul stands for unordered list.
  • li stands for list item, and you can have as many as you want.
  • Save often, and check for unexpected bugs by opening this file as a webpage.

CSS

  • A margin indicates how much space we want around the outside of an element.
  • A padding indicates how much space we want around the content inside an element.
  • The period preceding ".card" designates this as a class selector. Any element assigned to that class will gain these property assignments from the CSS.
  • You can adjust width, margin, border, box-shadow, and more. The values that follow operate in "top to bottom", and then "left to right" order.
  • Assigning colors can be used with "semantic" references, such as "grey", "blue", etc. But you can also use a hex# found externally online.

Git

  • git status: checks what branch we are currently in.
  • git checkout -b branch-name: creates a new branch, or switches to it.
  • pwd: print working directory.
  • ls: shows list of files and folders in working directory.
  • mkdir: make/create a directory/folder.
  • touch: creates a file in the directory.
  • start: opens a file.
  • git add -A:
  • git commit -m: sets a commit message for people to see.
  • git pull origin main: checks uf branch is up to date.
  • git status: checks for changes/unpushed changes in the active branch.
  • git checkout -b "branch name": specifies which branch you want changes applied to.

JavaScript

  • A variable is a named container that allows us to store dada in our code.
  • Control flow is the order in which a computer executes code in a script (Top to Bottom), and conditional statements can be used to interrupt "control flow".
  • Can name a variable basically anything -- though there are some restrictions.
  • JavaScript is CaSe-SeNsItIvE. "myVariable" and "myvariable" are not the same variable.
  • Everything in Javascript is an object, and can be stored in a variable.
  • Functions are reusable blocks of code that contain instructions for a specific task.
  • Math.floor() and Math.random() are Match "objects" that can be used together to create a random number.
